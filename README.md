# OOD Detection Evaluation (ODIN + ReAct)

This repository contains two Out-of-Distribution (OOD) detection evaluation pipelines:

1. **ODIN**
2. **ReAct**

Both pipelines compute:
- AUROC
- FPR@95TPR
- Inference latency
- Decision latency (ReAct only)


### Metrics
- AUROC
- FPR@95TPR
- Latency (ms per image)
